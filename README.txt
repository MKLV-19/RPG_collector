运行所需python版本：Python 3.8

操作步骤：
1、补全"RPG统计表.xlsx"中的信息，在“图片”列插入图片，修改“评分”栏的数据等
（注：命名该统计表时必须包含【统计】字段）
2、补全"文字说明.txt"中的信息，这些信息最终会在生成图片的底部显示
（注：命名该文档时必须包含【说明】字段）
3、（可跳过）打开"config.ini"修改图像生成的参数
4、在当前路径下打开cmd，运行"setup.bat"，并等待环境和依赖配置完成
（这一步可能出现依赖安装失败的报错，需要多运行几次，或者修改bat文件，使用国内镜像源来安装）
5、在虚拟环境下运行"python RPGcollector.py"，等待程序运行完成
6、在当前文件夹下生成的"output"目录中查看生成的图片
7、在cmd中输入"deactivate"，关闭虚拟环境
8、完成运行
